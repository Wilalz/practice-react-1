styles.module.css

Es una manera alternativa de manejar css por modulos y evitar que el codigo se pise así mismo, es decir que no hayan conflictos por selectores

Debemos crear un archvio llamado styles.module.css que tiene adentro el CSS con todas sus definiciones, por ejemplo, un selector de clase para una tarjeta .card

Ahora en el componente importar el style así import styles from "./styles.module.css" de esta manera se vincula el css y se pude utilizar para definir los selectores en los elemtos html del componente, ejemplo,

así es como dinamicamente se le trasfiere el selector .card al div

internamente, module le asigna un hash a cada elemento html para que no hayan conflictos -- Ver ejercicio 1 --
