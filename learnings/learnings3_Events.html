React

Eventos

Se refiere a las acciones que tiene un usuario en una app, las aciones del mouse (click, scroll) y del teclado (pulsacion de tecla) sobre los elementos de la pagina web. Lo que buscamos es manejar esos eventos y reaccionar para llevar a cabo una funcion asociada

Tipos de eventos

  • Nativos del navegador JS Vanila (mouse, teclado) addEventListener("mouseover", (event) => {});
  • Sinteticos (SyntheticEvents) React crea una capa sobre los eventos nativos para darle soporte a los eventos de todos los navegadores <button> onClick={handlerOnClick} </button>
  • Personalizados, creados por un dev

Nota: siempre, despues de un evento viene una funcion!

Si la funcion es corta se puede poner directamente onClick={setName="Pedro"}

Si se llama una funcion... si se ponen los parentesis al final haráq que se ejecute automaticamente la funcion onClick={funcionIncrementar()}

Si la funcion es muy larga, lo recomentable es poner un {handlerOnClick} que es una referencia a una funcion, es un llamado a esa funcion, que estará definida dentro del componente

normalmente en un imput buscamos optener el value del target (elemento) del evento {(e) => e.target.value}

Ejemplos

onChange - sirve para leer cambios de un input en un formulario

onClick - para reaccionar cuando se dá un click
